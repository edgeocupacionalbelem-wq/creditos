Dashboard de Créditos com Cloudinary.

- até 2 comprovantes por crédito
- identidade do crédito: recibo + id do pix
- aceita PDF, JPG, JPEG e PNG
- mantém comprovantes se o crédito continuar existindo
- apaga automaticamente se o crédito sumir da planilha nova

Variáveis no Render:
- DATABASE_URL
- CLOUDINARY_CLOUD_NAME
- CLOUDINARY_API_KEY
- CLOUDINARY_API_SECRET
